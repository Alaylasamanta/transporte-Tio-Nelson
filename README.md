
Transporte Escolar 
Objetivo: Divulgação
Status do Projeto: Em Desenvolvimento

Este é um site de divulgação moderno e responsivo, desenvolvido para apresentar o trabalho do transportador de forma impactante.
O foco principal foi criar uma experiência de usuário (UX) fluida, utilizando componentes do Bootstrap 5 e interatividade com JavaScript

---
## Funcionalidades Principais
*  **Design Responsivo**: Adaptado para Desktop, Tablet e Mobile com Bootstrap 5.
*  **CTAs de Alta Conversão**: Botões de contato vinculados diretamente ao WhatsApp da empresa.
*  **Interatividade**: Scripts em JavaScript para navegação fluida e validações.
*  **Mensagens Dinâmicas**: Link de API do WhatsApp configurado para facilitar o início do atendimento ao cliente.
